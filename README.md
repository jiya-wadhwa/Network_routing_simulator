# Network Routing Simulator

A simple Python project that finds shortest paths in a network using graph algorithms.

## Features

* Dijkstra’s Algorithm
* Bellman-Ford Algorithm
* User input for source and destination
* Displays path and cost

## How to Run

Run the file and enter nodes when prompted.

## Example

Input: A to E
Output: A -> C -> B -> E

## Concepts Used

* Graphs (Adjacency List)
* Shortest Path Algorithms
* Negative weight cycle analysed
